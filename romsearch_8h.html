Performs search for connected DS18B20 sensors. Discovered sensors' ROM addresses are returned in an array.

Parameters

directionA pointer to the port direction register
portinA pointer to the port input register
maskA bit mask, determining to which pin the sensor is connected
romcntA pointer to a variable when discovered sensor count will be written
romsA pointer to an array where discovered sensors' ROM addresses shall be stored
